# Portafolio Backend

Portfolio para el curso de Desarrollador Web Full Stack de Argentina Programa. 
En este repositorio se encuentra el Backend del proyecto. 
El Frontend se encuentra en el siguiente repositorio https://github.com/ND-magyp/ND_FrontEnd
El portfolio web se encuentra en: https://portfoliondf.web.app/index



Se desarrollo en las siguientes etapas:

a. Se desarrollo la parte de backend en forma local.

b. Se probo la implementación con el frontend

c. Una vez que frontend y backend funcionaron integrados en forma locar se realizó el despliegue en **Render**.



# Tecnologías

* Java: 19.0.2; Java HotSpot(TM) 64-Bit 
* Apache NetBeans IDE 17
